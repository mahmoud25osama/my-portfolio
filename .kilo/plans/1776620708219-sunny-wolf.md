## Phase 2: Add React-Bits Components (New Library)

React-Bits is a 38k+ star library with 110+ copy-paste animated components. It uses shadcn-style CLI installation — you copy component files into your project (not an npm dependency). This keeps the bundle lean.

### Installation Approach
React-Bits components are installed via:
```bash
npx shadcn@latest add @react-bits/<ComponentName>-TS-TW
```
Or by manually copying the component source files into the project.

## Phase 3: Hero Section Enhancements

### 3.1 Replace Typewriter with ScrambledText (react-bits)
**Current:** Custom `setTimeout`-based typewriter with stale-closure risk and direct `innerHTML` manipulation.

**New:** Use react-bits `ScrambledText` — a cyberpunk-style text scramble effect that randomizes characters before revealing the final text. Perfect for a developer portfolio.

**Implementation:**
- Install ScrambledText component from react-bits
- Replace the typewriter `useEffect` chains with `<ScrambledText>`
- Apply to the 3 rotating phrases under the hero name
- Keep the cycling behavior but use ScrambledText's `revealDuration` and `characters` props

### 3.2 Add Aurora Background (react-bits)
**Current:** CSS animated orbs (`anim-orb-1/2/3` keyframes) — basic floating gradient blobs.

**New:** Use react-bits `Aurora` component — a WebGL-powered aurora borealis background with flowing colors.

**Implementation:**
- Install Aurora component from react-bits
- Replace the 3 orb `<div>` elements in Hero with `<Aurora>`
- Configure with indigo/cyan color scheme: `colorStops={['#615fff', '#06b6d4', '#3b82f6']}`
- Set subtle amplitude (0.5-0.8) so it doesn't distract from content
- Wrap in the existing `absolute inset-0` container

### 3.3 Add Magnet Effect to CTA Buttons (react-bits)

**Implementation:**

## Phase 9: Navbar & Global Enhancements

### 9.1 Add Scroll Progress Indicator
**Current:** No visual feedback of scroll progress.

**New:** Add a thin gradient progress bar at the very top of the page that fills as the user scrolls.

**Implementation:**
- Create `src/components/layout/ScrollProgress.tsx`
- Uses `window.scrollY` / `document.body.scrollHeight` to calculate percentage
- Render as a fixed `<div>` at `top: 0` with `z-index: 60`
- Gradient from indigo to cyan (matching theme)
- Thin: 2-3px height
- Animate width with CSS transition or GSAP

### 9.2 Add Command Palette (cmdk)
**Current:** Only navigation is via navbar links or scrolling.

**New:** Add a keyboard-triggered command palette (Ctrl+K) for quick navigation, theme toggle, and actions.

**Implementation:**
- Install `cmdk` npm package
- Create `src/components/layout/CommandPalette.tsx`
- Commands: Navigate to each section, Download CV, Open GitHub, Toggle theme
- Keyboard shortcut: `Ctrl+K` or `Cmd+K`
- Style with glassmorphism to match portfolio theme
- Add to App.tsx

### 9.3 Improve CustomCursor
**Current:** DOM queries for interactables run only at mount time — missing dynamically added elements.

**New:** Use event delegation on `document` instead of querying specific elements.

**Implementation:**
- Replace `document.querySelectorAll('a, button, [data-cursor]')` with event delegation
- Listen for `mouseover`/`mouseout` on `document` and check `e.target.closest('a, button, [data-cursor]')`
- This automatically handles dynamically rendered interactables (e.g., after project filter change)

## Phase 10: Performance & Architecture Fixes

### 10.1 Centralize GSAP ScrollTrigger Registration
**Current:** `gsap.registerPlugin(ScrollTrigger)` is called in 5 separate section files.

**New:** Register once in `main.tsx` before the app renders.

### 10.2 Use `.will-reveal` CSS Class Instead of Inline `style={{ opacity: 0 }}`
**Current:** 20+ elements use `style={{ opacity: 0 }}` as initial GSAP state.

**New:** Use the `.will-reveal` class already defined in `index.css` (line ~380) but never used.

### 10.3 Fix Logo SVG ID Collisions
**Current:** SVG gradient IDs (`moGradient`, `iconGradient`) clash when Navbar and Footer both render on the same page.

**New:** Use React 19's `useId()` hook to generate unique IDs per component instance.

### 10.4 Remove `'use client'` Directives
**Current:** 8 components have `'use client'` which is a Next.js convention with no effect in Vite.

**New:** Remove all `'use client'` directives.
